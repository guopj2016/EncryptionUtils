# EncryptionUtils
## 说明
加密工具类，这里收集了一些常用的加密方法，包括AES，DES，3DES，MD5，RSA，SHA1，SHA256，SHA384，SHA512。
## 使用
### 依赖（审核中）
```
compile 'com.wuyifeng:EncryptionUtils:1.0.0'
```
